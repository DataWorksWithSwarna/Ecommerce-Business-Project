# Ecommerce Buisness Sales~

## Project Overview

The Ecommerce Sales Dataset captures transactional-level information from an online retail platform over a defined time period. 
Each record represents a unique customer order and contains details covering products, customers, pricing, sales performance, and purchasing behavior. 

## Key Objectives of the Project

- Analyze daily, monthly, and seasonal sales trends

- Identify top-selling products, categories, and brands

- Evaluate revenue, profit, quantity sold, and pricing patterns

- Develop interactive dashboards (SQL, Python, Power BI, Tableau)

## Skills Showcased

-  EDA with Python (Pandas, Seaborn, Matplotlib)
-  SQL querying (MySQL/PostgreSQL)
-  Power BI Sales Dashboards
-  Forecasting & ML models (Sales Prediction, Customer Segmentation)
